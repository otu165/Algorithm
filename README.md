## Categories

| 폴더 | 구성 |
|---|---|
| backtracking | 순열, 조합, 재귀 탐색 |
| bfs_dfs | BFS, DFS |
| binary_search | 이분 탐색 |
| dp | 동적 계획법 |
| graph | 최단 경로, MST, 그래프 이론 |
| greedy | 탐욕 알고리즘 |
| hash | 딕셔너리, 집합, 카운팅 |
| heap | 힙, 우선순위 큐 |
| implementation | 구현, 시뮬레이션 |
| math | 수학, 규칙 찾기 |
| regex | 정규표현식 |
| sort | 정렬 |
| stack_queue | 스택, 큐, 덱 |
| string | 문자열 처리, 파싱 |
